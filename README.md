# e-plantShopping

Welcome to the Paradise Nursery shopping application (e-plantShopping), an online shop offering a variety of house plants.

## Features
- A Landing page with a button linking to the product listing page
- A navigation bar with links to the landing, product listing, and shopping cart pages
- A product listing page categorized by plant types, each having unique cards with images, name, cost, and Add to cart functionality.
- A shopping cart page displaying selected products, subtotal for each item, and total global cost, along with item increment/decrement/remove actions.

Built with React and Redux.
